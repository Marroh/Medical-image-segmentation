#### 效果演示
点击`功能栏->导入->选择dcm序列文件中的一个dcm文件`。等待几秒钟可完全导入，用滚轮切换dcm切片。
![显示](result_imgs/显示.png)

点击`功能栏->导入->选择dcm序列文件中的一个dcm文件`，等分割结束后会在当前目录下生成每一层的.npy分割文件，之后再点击`功能栏->分割`。
